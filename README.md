# Masterlight IoT データサーバ開発
## 概要
脱毛機Masterlightのデータを収集するデータサーバアプリケーションを開発する。

## 目的
脱毛機のデータをサーバで集約し、保守サポート体制の強化、顧客サービスの向上を目指す。
